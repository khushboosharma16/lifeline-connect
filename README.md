# Lifeline Connect

A no-build, front-end prototype for a blood donation coordination service. It uses browser local storage for profile, availability, appointment, and help-request interactions.

## Publish on Vercel

Import this folder or repository in Vercel and use the default settings. No build command or environment variables are required; Vercel will serve `index.html` automatically.

For local preview, open `index.html` in a browser or run `npx serve .`.
